# Expense-Tracker-Data-Science
A fintech-style expense tracker dashboard using Python, Streamlit, and data visualization with synthetic data.


Overview
Track and analyze personal expenses using Python.

Problem
People don’t know where their money goes.

 Features
- Category analysis
- Monthly trends
- Visualization

Tech Stack:

Python, Pandas, Matplotlib, Seaborn


Key Insights:

Identified top spending categories
Tracked monthly expense trends
Detected overspending patterns
